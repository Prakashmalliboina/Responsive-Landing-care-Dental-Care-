DentalCare - Painless Root Canal Treatment Landing Page

A modern, responsive, and high-conversion landing page designed for a dental clinic specializing in Root Canal Treatments (RCT). This project showcases a clean user interface with sections for services, patient testimonials, clinic locations, and appointment booking.
